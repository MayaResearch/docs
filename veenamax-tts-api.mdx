## Overview

VeenaMAX is a high-performance, advanced Text-to-Speech API that converts text into natural-sounding speech with emotional intelligence and blazing-fast response times. It can speak English and Hindi - the most widely used languages in India.

## API Reference

### Authentication

<Note>
  All requests require authentication using a Bearer token in the Authorization header.
</Note>

```
Authorization: Bearer YOUR_API_KEY_HERE
```

### Generate Speech

<Endpoint method="POST" api="/generate" />

Generates high-quality speech audio from the provided text input with ultra-low latency.

#### Request Headers

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token format: `Bearer <API_KEY>`
</ParamField>

#### Request Body

<ParamField body="text" type="string" required>
  The text to convert to speech (supports Hindi, English, Hinglish)
</ParamField>

<ParamField body="speaker_id" type="string" required>
  Voice identifier (see available voices below)
</ParamField>

<ParamField body="streaming" type="boolean" default="false">
  Enable streaming audio generation for real-time applications
</ParamField>

<ParamField body="normalize" type="boolean" default="true">
  Enable intelligent text normalization
</ParamField>

<CodeGroup>

```bash cURL
curl --location 'https://api.mayaresearch.ai/generate' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_API_KEY_HERE' \
--data '{
  "text": "If you don'\''t stop and look around once in a while, you could miss it.",
  "speaker_id": "varun_chat",
  "streaming": true,
  "normalize": true
}'
```

```javascript JavaScript
const response = await fetch('https://api.mayaresearch.ai/generate', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY_HERE'
  },
  body: JSON.stringify({
    text: "If you don't stop and look around once in a while, you could miss it.",
    speaker_id: "varun_chat",
    streaming: true,
    normalize: true
  })
});
```

```python Python
import requests

url = "https://api.mayaresearch.ai/generate"
headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY_HERE"
}
data = {
    "text": "If you don't stop and look around once in a while, you could miss it.",
    "speaker_id": "varun_chat",
    "streaming": True,
    "normalize": True
}

response = requests.post(url, json=data, headers=headers)
```

</CodeGroup>

## Error Handling

<Warning>
  Always implement proper error handling in your application to gracefully handle API errors.
</Warning>

| Error Code | Description | Resolution |
|------------|-------------|------------|
| <Icon icon="circle-exclamation" color="red" /> **400** | Invalid request format | Check JSON syntax and required fields |
| <Icon icon="lock" color="orange" /> **401** | Authentication failed | Verify API key and Bearer token format |
| <Icon icon="ban" color="red" /> **403** | Access denied | Check API permissions and quotas |
| <Icon icon="clock" color="yellow" /> **429** | Rate limit exceeded | Implement exponential backoff |
| <Icon icon="server" color="red" /> **500** | Internal server error | Contact support if persistent |

<Tip>
  **Pro tip:** Use exponential backoff with jitter for handling rate limits and transient errors effectively.
</Tip>
