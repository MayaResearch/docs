## API Reference

### Authentication

<Note>

</Note>

```
X-API-Key: maya_YOUR_API_KEY_HERE
```

## How It Works

Choose a voice character and generate speech with a simple API call:

```json
{
  "voice_id": "Ava",
  "text": "Welcome back to another episode of our podcast! Today we are diving into an absolutely fascinating topic.",
  "stream": false
}
```

<Note>
  Browse all available voices using the [Get Characters endpoint](/api-reference/endpoint/maya1-characters) to find the perfect voice for your needs.
</Note>
