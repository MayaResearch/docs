<Note>
  This endpoint generates high-quality speech audio from text input with ultra-low latency and support for Hindi, English, and Hinglish.
</Note>

## Request Body

<ParamField body="text" type="string" required>
  The text to convert to speech. Supports:
  - English: "Hello, how are you?"
  - Hindi (Devanagari): "आप कैसे हैं?"
  - Hindi (Roman): "Aap kaise hain?"
  - Hinglish: "Meeting 3 baje hai, please time pe aana"
</ParamField>

<ParamField body="speaker_id" type="string" required>
  Voice identifier to use for speech generation
  
  **Female Voices:**
  - `charu_soft` - Gentle storyteller
  - `keerti_joy` - Energetic presenter
  - `mohini_whispers` - ASMR specialist
  - `maitri_connect` - Professional narrator
  - `soumya_calm` - Relaxation expert
  - `vinaya_assist` - Virtual assistant
  
  **Male Voices:**
  - `varun_chat` - Authoritative speaker
  - `agastya_impact` - Motivational coach
</ParamField>

<ParamField body="streaming" type="boolean" default="false">
  Enable streaming audio generation for real-time applications
  - `true` - Stream audio chunks as they're generated
  - `false` - Return complete audio file after processing
</ParamField>

<ParamField body="normalize" type="boolean" default="true">
  Enable intelligent text normalization for numbers, abbreviations, and special characters
</ParamField>

## Streaming Response

When `streaming` is set to `true`, the API returns audio chunks as they are generated:

```javascript
const response = await fetch('https://api.mayaresearch.ai/generate', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer vna_master_D_z9rDcMb1b9woKT0Z4AH9xukkFUYgiQ4da2G2FyNF1'
  },
  body: JSON.stringify({
    text: "Long text for streaming...",
    speaker_id: "vinaya_assist",
    streaming: true,
    normalize: true
  })
});

const reader = response.body.getReader();
const audioChunks = [];

while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  audioChunks.push(value);
  
  // Process or play audio chunk immediately
  processAudioChunk(value);
}

// Combine all chunks for complete audio
const completeAudio = new Blob(audioChunks, { type: 'audio/wav' });
```

<Tip>
  Use streaming mode for:
  - Real-time voice assistants
  - Live customer support
  - Interactive voice applications
  - Long-form content where you want to start playback immediately
</Tip>
